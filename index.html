Petunjuk Penggunaan Aplikasi

1

Unggah File LPJ & Penalti

Unggah file unduhan dari sistem SAKTI untuk LPJ. Anda juga dapat mengunggah file Penalti (opsional) untuk menghitung distribusi penalti secara otomatis ke dalam data penilaian.

2

Proses dan Verifikasi

Klik tombol Proses Data. Sistem akan mendeteksi satker, rincian transaksi, dan mendistribusikan nilai sesuai aturan penilaian yang berlaku.

3

Monitoring & Dashboard

Gunakan Ringkasan Kinerja Satker untuk melihat tren penolakan, kepatuhan pajak, dan saldo kas secara visual. Dashboard ini dapat diunduh sebagai gambar untuk pelaporan.

4

Unduh Rekap Akhir

Setelah semua data terproses, unduh file CSV hasil akhir. File ini berisi rincian nilai yang sudah siap digunakan untuk pengisian portal penilaian atau distribusi ke satker.
